# v1.1.7 - 允许多用户并行同步

## 任务描述
修复自动同步时"其他用户正在同步，跳过"的问题，允许多个用户并行同步。

## 执行结果

### 变更内容
1. 移除 `getAnyUserSyncing` import
2. 删除 `executeUserSync` 函数中的全局锁检查逻辑（原第 76-80 行）

### 效果
- 修改前：当用户 A 正在同步时，用户 B/C/D 的定时同步会被跳过
- 修改后：多个用户可以同时并行同步，但同一用户不能重复同步
