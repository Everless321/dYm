# 修复 macOS 托盘图标不显示问题

## 时间
2026-02-02 17:32:00

## 问题描述
用户报告在 macOS 上运行应用后，菜单栏中没有显示托盘图标。

## 原因分析
1. 使用了 512x512 的彩色应用图标作为托盘图标
2. 设置了 `setTemplateImage(true)`，但 template 模式需要黑白透明图标
3. 彩色图标在 template 模式下无法正确显示

## 解决方案
1. 创建专用托盘图标文件 `trayTemplate.png` 和 `trayTemplate@2x.png`
2. 移除 macOS template 模式设置（因为使用的是彩色图标）
3. 调整 macOS 托盘图标大小为 18x18（更适合菜单栏显示）
4. 为不同平台使用不同的图标和大小策略

## 修改文件
- `src/main/index.ts` - 更新 createTray 函数，添加专用托盘图标导入
- `resources/trayTemplate.png` - 新增托盘图标
- `resources/trayTemplate@2x.png` - 新增 Retina 托盘图标
