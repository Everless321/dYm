# 修复视频解析下载功能

## 任务描述
修复视频/图集链接解析下载功能无法正常工作的问题。

## 问题分析
`video:getDetail` handler 中，错误地将 `fetchVideoDetail` 返回的 `PostDetailFilter` 类实例当作原始 API 响应数据处理。

### 错误的代码
```typescript
const data = detail as any
const coverUrl = data.cover?.urlList?.[0] || data.dynamicCover?.urlList?.[0] || ''
videoUrl: data.video?.playAddr?.urlList?.[0] || ''
imageUrls: data.images?.map((img) => img.urlList?.[0])
```

### 问题原因
`PostDetailFilter` 是 dy-downloader 库中的 Filter 类，其属性是经过处理的 getter 方法，直接返回可用的 URL 字符串，而非原始嵌套的 `urlList` 结构。

## 解决方案
直接使用 `PostDetailFilter` 的 getter 方法获取已处理好的 URL：

```typescript
const coverUrl = detail.cover || detail.animatedCover || ''
videoUrl: detail.videoPlayAddr?.[0] || ''
imageUrls: detail.images || []
```
