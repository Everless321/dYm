# Web Migration - Electron to Fullstack Web App

**Date**: 2026-02-09
**Branch**: feat/web-migration

## Task Description
Migrate the existing Electron desktop app (dYmanager) to a fullstack web application with:
- **Backend**: Fastify + better-sqlite3 + WebSocket + node-cron
- **Frontend**: Vite + React 19 + shadcn/ui + Tailwind + TanStack Query
- **Structure**: pnpm Monorepo (packages/shared, packages/server, packages/web)

## Scope
- Remove Electron-specific features (cookie browser fetch, tray, clipboard detection, auto-updater)
- Move all services (syncer, downloader, scheduler, analyzer) to Fastify backend
- Replace IPC with REST API + WebSocket
- Adapt frontend to use HTTP/WS instead of window.api
