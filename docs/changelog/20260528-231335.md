# 20260528-231335 — 网页端播放器进度条 + 键盘 seek + 流稳定性

## 背景
Issue #6 反馈：长视频在网页端浏览时
1. 原生缓冲条点击未加载位置会卡死，关闭后再打开也无法播放
2. 没有进度条 / 快进快退，看长视频只能等

## 待办

- [x] 进度条点击 / 拖动 seek
- [x] 键盘快捷键：← / → seek ±5s，空格播放暂停，↑/↓ 切上下条
- [x] 视频 `error` / `stalled` 容错：自动恢复
- [x] 后端流：client abort 时销毁读流，避免堆积
- [x] 资源版本号 bump（app.js?v=5）
- [x] package.json 版本 1.8.6
