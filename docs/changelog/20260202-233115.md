# v1.1.5 - 剪贴板检测功能

## 任务描述
实现剪贴板检测功能：当用户切换回应用窗口时，自动检测是否复制了抖音分享链接，并提示添加到用户管理。

## 新增功能
- **剪贴板自动检测**：切换回应用窗口时自动检测抖音链接
- **一键添加用户**：检测到链接后弹出 toast 提示，点击即可添加
- **防抖处理**：500ms 防抖避免重复触发
- **冷却机制**：同一链接 30 秒内不重复提示
