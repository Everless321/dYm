# 用户管理筛选条件增强

**日期**: 2026-02-03 16:30:00

## 任务描述

用户反馈用户管理页面的搜索条件不够用，需要更多筛选维度。

## 实施方案

### 新增筛选条件

1. **首页显示** - 全部 / 是 / 否
2. **自动同步** - 全部 / 开启 / 关闭
3. **下载状态** - 全部 / 已完成 / 部分下载 / 未开始
4. **排序方式** - 默认 / 未下载数 / 总作品数

### 修改文件

**src/renderer/src/pages/settings/UsersPage.tsx**
- 新增筛选类型定义: `ShowInHomeFilter`, `AutoSyncFilter`, `DownloadStatusFilter`
- 新增筛选状态变量
- 更新 `filteredUsers` 逻辑，支持多条件组合筛选
- 添加筛选 UI 行，包含 4 个下拉选择器
- 更新用户计数显示: `(筛选后数量/总数量)`
- 筛选条件变化时自动重置到第一页
